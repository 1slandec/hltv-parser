# Название проекта / Тема исследования

Анализ связи стран происхождения игроков Counter-Strike с рейтингом лучших команд мира HTLV

# Объект исследования

Количество игроков в разных диапазонах рейтинга (TOP 10, 50, 100, 200), агрегированное по странам в разные года (2016-2026)

Динамика доминации различных регионов в киберспортивной дисциплине CS

# Задачи

1. Сбор данных - парсинг данных с сайта hltv.org/ranking/teams/
2. Предобработка и очистка данных - определение региона каждого игрока (СНГ, Европа и др.)
3. Анализ данных - 
4. Визуализация - построение графика временного ряда для каждого региона
